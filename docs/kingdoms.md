
# گاه‌شمار پادشاهی‌های ایران

* گاه‌شمارهای ایجاد شده براساس اطلاعات [ویکی‌پدیا فارسی](https://w.wiki/ALhg) می‌باشد.
* تاریخ‌های درج شده در گاه‌شمارها براساس **میلادی** می‌باشد.
